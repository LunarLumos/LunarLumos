   - **Telegram Notifications:** Stay informed with instant updates 📲.
   - **Customizable Intervals:** Tailor your scan schedules ⏰.
   - **Color-Coded Output:** Enjoy clear, vibrant terminal feedback 🌈.

   **Why NetWarn?**
